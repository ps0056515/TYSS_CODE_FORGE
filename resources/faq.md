# CodeForge FAQ

## Getting Started

**Q: How do I create an account?**
A: Click "Sign In" in the top navigation. You can sign in with Google, GitHub, or create an account with your email and password.

**Q: Is CodeForge free to use?**
A: Yes, the core features (practice problems, MCQs, leaderboard) are free. A Pro plan with additional features may be available in the future.

**Q: What programming languages are supported?**
A: Python 3, JavaScript (Node.js), TypeScript, Java, C++, C, Go, and Rust.

## Practice Problems

**Q: How do I submit a solution?**
A: Open a problem, write your code in the editor, and click the "Submit" button. Your code will be tested against all test cases.

**Q: Why is my code timing out?**
A: Code execution has a 2-second time limit. Optimize your algorithm — look for O(n²) loops that could be O(n log n) or O(n).

**Q: Can I see other users' solutions?**
A: After solving a problem, you can view the editorial and discussion section.

**Q: How do I run my code without submitting?**
A: Use the "Run" button to test with sample inputs. Use "Run Samples" to test against all provided sample test cases.

## Code Editor

**Q: How do I change the font size in the editor?**
A: Right-click in the editor for options, or use Ctrl+= / Ctrl+- to zoom.

**Q: Does the editor support autocomplete?**
A: Yes! The Monaco editor provides IntelliSense for supported languages.

**Q: Can I use keyboard shortcuts?**
A: Yes, standard Monaco editor shortcuts work: Ctrl+Z (undo), Ctrl+/ (comment/uncomment), Alt+Shift+F (format), etc.

## Submissions & Progress

**Q: Where can I see my submission history?**
A: Go to your Profile page or click "Submissions" in the navigation.

**Q: How is the leaderboard score calculated?**
A: Points are earned by solving problems. Easy = 10pts, Medium = 25pts, Hard = 50pts. Contest wins earn bonus points.

**Q: Can I reset my progress?**
A: Contact admin support to reset your account progress.

## Contests

**Q: How do I join a contest?**
A: Navigate to the Contests page and click "Register" on an upcoming contest.

**Q: What happens if I submit after the contest ends?**
A: Late submissions are not counted for contest scoring but still count for practice.

## Technical Issues

**Q: My code runs locally but fails on CodeForge?**
A: Check for: (1) hardcoded file paths, (2) missing input parsing, (3) library imports not available, (4) time/memory limits exceeded.

**Q: The editor is not loading?**
A: Try refreshing the page. Ensure JavaScript is enabled and you're using a modern browser (Chrome, Firefox, Edge, Safari).

**Q: I'm getting a CORS error?**
A: Make sure you're accessing the platform via the correct URL (http://localhost:3002 for local dev).

## Projects

**Q: How do I submit a project?**
A: Navigate to Practice > Projects, select a project, and upload your solution as a ZIP file.

**Q: What should my ZIP file contain?**
A: Include all source files, a README.md explaining your approach, and any required configuration files. Do not include node_modules or build artifacts.

## Account & Settings

**Q: How do I change my theme?**
A: Click the sun/moon icon in the top-right navigation to toggle between light and dark mode.

**Q: Can I link multiple OAuth providers?**
A: Currently, each account supports one OAuth provider.

**Q: How do I become an admin?**
A: Admin access is granted by the platform administrator. Contact your trainer or organization admin.
