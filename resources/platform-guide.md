# CodeForge Platform Guide

## What is CodeForge?
CodeForge is a modern coding practice and learning platform similar to LeetCode and CodeChef. It lets you practice coding problems, take MCQ quizzes, work on projects, participate in contests, and track your progress on a leaderboard.

## Features

### Practice Problems
- Browse hundreds of coding problems organized by category, language, and difficulty
- Solve problems in an in-browser Monaco code editor
- Supported languages: Python, JavaScript, TypeScript, Java, C++, C, Go, Rust
- Get instant feedback with test case results
- Track your submission history

### MCQ Quizzes
- Multiple-choice questions organized by technology (Python, JavaScript, React, etc.)
- Test your theoretical knowledge alongside coding skills

### Projects
- Work on real-world project assignments
- Submit your work as a ZIP file
- Projects are organized by category (Web Dev, Data Science, etc.)

### Courses
- Browse a catalog of courses
- Structured learning paths

### Contests
- Participate in timed coding contests
- Live scoring and rankings

### Leaderboard
- See top performers globally
- Filter by language, category, or time period

## How to Use the Code Editor
1. Navigate to Practice > select a problem
2. Choose your programming language from the dropdown
3. Write your solution in the Monaco editor
4. Click "Run" to test with sample inputs
5. Click "Submit" to evaluate against all test cases
6. View results and submission history

## Authentication
- Sign in with Google or GitHub OAuth
- Or use email/password login
- Sessions last 30 days

## Admin / Trainer Features
- Admins can manage problems, courses, and contests
- Access the Admin panel via the top navigation
- Admin status is granted to users with "admin" or "trainer" prefix in username, or via CODEFORGE_ADMINS env var

## Supported Programming Languages
- Python 3
- JavaScript (Node.js)
- TypeScript
- Java
- C++
- C
- Go
- Rust

## Difficulty Levels
- Easy: Beginner-friendly problems
- Medium: Intermediate challenges
- Hard: Advanced algorithmic problems

## Scoring & Points
- Easy problems: 10 points
- Medium problems: 25 points
- Hard problems: 50 points
- Bonus points for first-time solves and contest wins
